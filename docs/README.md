## 📝 Convenciones

### Nombres de Archivos
- Usar MAYÚSCULAS para documentos importantes
- Usar snake_case o kebab-case
- Incluir fecha en documentos de correcciones (ej: `FIX_2026_02_20.md`)

### Estructura de Documentos
1. Título claro
2. Descripción breve
3. Problema/Contexto
4. Solución
5. Pasos de implementación
6. Pruebas
7. Notas adicionales
